# 스토리지 대안 가이드

## 문제

Cloudflare R2는 결제 정보 등록이 필수입니다. 자동 결제를 원하지 않는 경우 대안이 필요합니다.

## 대안 옵션

### 옵션 1: Render 파일 시스템 사용 (권장, 완전 무료)

**장점:**
- ✅ 결제 정보 불필요
- ✅ 완전 무료
- ✅ 추가 설정 불필요
- ✅ Render와 통합

**단점:**
- ⚠️ 서버 재시작 시 파일 손실 가능 (영구 볼륨 사용 가능)
- ⚠️ 백업 필요

**구현:**
- Render의 파일 시스템에 직접 저장
- 영구 볼륨 마운트 가능 (무료 플랜에서도 가능)

### 옵션 2: AWS S3 무료 플랜 (12개월 무료)

**장점:**
- ✅ 12개월 무료 (5GB 스토리지)
- ✅ 사용량 알림 설정 가능
- ✅ 결제 정보 필요하지만 알림으로 제어 가능

**단점:**
- ⚠️ 결제 정보 등록 필요
- ⚠️ 12개월 후 유료 전환

### 옵션 3: Google Cloud Storage 무료 플랜

**장점:**
- ✅ 5GB 무료 스토리지
- ✅ 사용량 제한 설정 가능

**단점:**
- ⚠️ 결제 정보 등록 필요

### 옵션 4: Backblaze B2

**장점:**
- ✅ 10GB 무료 스토리지
- ✅ 저렴한 가격

**단점:**
- ⚠️ 결제 정보 필요할 수 있음

## 추천: Render 파일 시스템 (옵션 1)

현재 사용량(300MB/월)을 고려하면 Render 파일 시스템으로 충분합니다.

### 구현 방법

1. **Render 영구 볼륨 사용** (무료 플랜에서도 가능)
2. **파일을 Render 서버에 직접 저장**
3. **정적 파일 서빙으로 이미지 제공**

### 장점

- 결제 정보 불필요
- 완전 무료
- Render와 완벽 통합
- 설정 간단
